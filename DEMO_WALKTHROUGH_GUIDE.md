# Barfliz Interactive Demo Guide

## Accessing the Demo

Users can access the full interactive demo in two ways:

1. **From the onboarding screen**: Click the "Watch Interactive Demo" button
2. **Direct URL**: Navigate to `/demo`

## Demo Features

The demo provides a complete 12-step walkthrough of the entire Barfliz user experience:

### Step 1: Sign Up & Authentication
- Phone number verification via Twilio SMS
- OTP code validation
- Age gate (21+ verification)
- Terms of service

### Step 2: Profile Setup
- Name and birthday
- Profile photo upload
- Drink preferences selection
- Vibe tags (Dance Floor, Rooftop, Dive Bars, etc.)
- Home city selection

### Step 3: Permissions Request
- Location services (for nearby venues)
- Push notifications (for messages & alerts)
- Contacts access (for finding friends)
- Radar geofencing setup

### Step 4: Home Dashboard
- Tonight Status setter (Staying In / Going Out Soon / Out Now)
- Friends who are out tonight
- Tonight Feed with friend locations
- DD Mode toggle (Designated Driver)
- Weather card
- Quick actions: Create Swarm, Check In, Message

### Step 5: Interactive Map
- 682+ venues (Darwin, South Florida)
- Real-time friend locations
- Venue filters (Bars, Clubs, Lounges, Breweries)
- Search venues by name
- Radius control
- Venue details: hours, address, crowd size
- Google Places integration

### Step 6: Tonight Status & Social Feed
- "Out Now" friends with live locations
- "Going Out Soon" friends list
- Current venue for each friend
- DD badges for designated drivers
- Vibe tags visible
- One-tap to message or view profile

### Step 7: Friends & Connections
- Send/accept friend requests
- Search users by name or username
- Import contacts to find friends
- View friend profiles
- Block/unblock users
- Friend activity history

### Step 8: Messaging
- One-on-one conversations
- Real-time message delivery
- Read receipts
- Share location in messages
- Share music (Spotify integration)
- Send virtual gifts & drinks
- Emoji reactions

### Step 9: Swarms (Group Plans)
- Create swarm with name, date, time
- Invite multiple friends
- Set meeting venue
- RSVP tracking (Going, Maybe, Invited)
- Group chat in swarm
- Share swarm details
- Date filter for upcoming swarms

### Step 10: Payments & Splitting
- Link Venmo/PayPal/Cash App
- Send payment to friends
- Request payment
- Group bill splitting
- Payment history
- QR code for receiving

### Step 11: Virtual Gifts
- Gift catalog: drinks, shots, champagne
- Send gifts with messages
- Gift inbox to receive items
- Gift history tracking
- Fun animations

### Step 12: Profile & Settings
- Edit profile details
- Change avatar
- Update drink preferences
- Privacy settings (ghost mode)
- Notification preferences
- Payment provider settings
- Safety features (safe arrival, emergency contacts)
- Language selection (i18n ready)
- Help center & support

## Demo Controls

- **Navigation**: Use Previous/Next buttons to move between steps
- **Progress Bar**: Visual indicator showing completion percentage
- **Step Dots**: Click any dot to jump to that step
- **Auto-play**: Coming soon - automatic progression through steps
- **Close**: X button in top-right to exit demo
